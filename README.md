# **README.md**


## **自己紹介**


私は特に Pyhon3 を得意とする 1994 年生まれの日本人の WEB エンジニアです。


自身の価値を最も発揮できるのは 0 → 1 フェーズでの開発だと思っています。


得意なことは以下のとおりです。


* Python3 がコアスキルです。
* 並列処理、非同期処理を活用した 1,000 万件以上の大規模クエリを伴う演算処理が得意です。
* 上記と同様の複数コンテナを利用した大規模なクローラ作成が得意です。
* WEB アプリの制作はバックエンド側が得意。フロントもある程度書けます。
* OAuth, JWT, Session・Cookie 全ての認証を作成することが出来ます。
* 大規模なデータの加工や Elastic Search を用いた全文検索等の設計・開発を一貫した開発が可能です。

## **経歴**


### **2022 年 ~ 現在(勤務先で WEB エンジニア/正社員 | 自身の起業した会社で社長 )**


2022 年の 7 月に新しい会社に正社員の WEB エンジニアとして就職しました。


- ビッグデータの解析、及びそのデータの可視化を行う WEB アプリケーションの開発を現在主に担当しています。


- バックエンドには `Django`, フロントエンドには `React(JavaScript)` , 全文検索に `Elastic Search` を使用しています。


- 小規模組織であるため、設計から開発まで全てを 1 人で担当。ただし、アクティブユーザーが少ないため保守はほとんど行っていません。


- 一部の業務において社内向けの RPA アプリケーションの開発も平行して行っています。(API 等を用いて特定業務を自動化、等)


- 使用言語は `Python3`, `JavaScript` です。


### **2020 年 ~ 2021 年: 自身で株式会社を開業 (本社: 奈良県 | 社長)**


自社サービスの開発に取り組むとともにいくつかの業務委託案件に携わりました。


- Django を用いたサーバーサイドレンダリングのアプリケーションを構築しました。


- Django, React を用いた API 通信を行う WEB アプリケーションを構築しました。


- Echo(Go) を用いたサーバーサイドレンダリングのアプリケーションを構築しました。


- 複数のプロキシを用いた非同期通信クローリングアプリの開発を行いました。


- Windows 向け小規模アプリケーションの開発にも携わりました。


- 使用言語は `Pyhon3`, `Go`, `JavaScript` です。


### **2020 年: 新卒入社した人材紹介会社を自己都合にて退職。フリーランスに転身。**


退職後はフリーランスとして、クローラーの作成、 Chrome Extentions の開発等の 一人で完結可能な小規模なアプリケーションの開発代行を行っていました。


- 使用言語は `Python3`, `JavaScript` です。


- その他、採用代行やテレホンアポイントメントの代行の案件を行っていました。(人事/広報領域がメイン)
